# docs/

Hosts the demo media referenced from the top-level README.

| Path | Notes |
|------|-------|
| `demo.gif` | Hero animation at the top of the README. Aim for ≤ 8 MB so GitHub renders it inline. Record with [ScreenToGif](https://www.screentogif.com/) or [LICEcap](https://www.cockos.com/licecap/). |
| `screenshots/main.png` | Static screenshot of the main window — used after the Features section. |

Both files are intentionally **not** generated by CI; they're checked in
manually so the README looks right on github.com without a build step.
